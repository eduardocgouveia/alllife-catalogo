# Catálogo de Serviços e Produtos · ALL LIFE INSTITUTE

> Sistema operacional do ALL LIFE INSTITUTE · catálogo navegável de especialistas, jornadas, vendas e voz da marca. Hospedagem estática em GitHub Pages.

## Estrutura

```
.
├── index.html                              ← redirect para o catálogo
├── CATALOGO-DE-PRODUTOS-E-SERVICOS.html    ← catálogo navegável principal
├── MANIFESTO.html                          ← manifesto V9 íntegra
├── LINHAS.html                             ← anexo · linhas de produto
├── QUESTIONARIO-ESPECIALISTAS.html         ← formulário de coleta
└── assets/
    ├── fonts/    Monument Grotesk · Simon Mono
    ├── images/   Portraits e ambientes
    └── logos/    DS 3.0 oficial
```

Cada arquivo é standalone · pode ser aberto direto no navegador.

## Formulário de especialistas · envio por email

O `QUESTIONARIO-ESPECIALISTAS.html` envia as respostas + fotos via **FormSubmit.co** para `eduardo@agenciaatacama.com.br`.

**Ativação inicial (uma vez só):**

1. Preencher e enviar o formulário pela primeira vez
2. FormSubmit envia email de confirmação ao Eduardo
3. Clicar no link de ativação no email
4. A partir daí, todos os envios chegam diretamente

**Limites do plano gratuito:**

- Anexos até ~10 MB no total por envio
- 50 envios/mês
